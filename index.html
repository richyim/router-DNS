Router DNS & Gemini Image Analyzer

This is a desktop application combining a Python Tkinter GUI with the Google Gemini API to analyze images.

How to Use:

  1. Clone or download this repository.
  2. Open the script and paste your YOUR_GEMINI_API_KEY.
  3. Run the Python file, select a JPG image, and view the AI's analysis in a pop-up window!
